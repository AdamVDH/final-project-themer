Themer Site

This is my final project for Web 103. This page showcases my "themer," as I call it. It's a theme changer, changing the CSS of this page by the press of a button. It's capable of more than I've showcased, and it's easily expandable!

Below is a "notecard." I thought to use it like an index card.

Further below, in the header "Heading With Badge," is a badge that says "New!" I stole that idea and the general design from Bootstrap.
